# HTML Dummy
Creates a minimal valid HTML file
```
<!doctype html>
<html lang=en> 
<head>
<meta charset=utf-8>
<title>title</title>
</head>
<body><!-- content --></body>
</html>
```

## Install
```
npm install -g
```

## Usage
```
html
```

Options:
```
--lang    Set html language    [string] [default: en]
--charset    Set meta charset    [string] [default: utf-8]
--title    Set title    [string] [default: title]
```
